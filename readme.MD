### Understanding Generative AI with Retrieval and Intelligent Agents

This repository is a **general exploration and reference space** for understanding **modern Generative AI systems**, with a focus on:

- Retrieval-Augmented Generation (RAG)
- Agentic AI and intelligent agents
- Multi-agent systems
- Graph-based agent workflows (LangGraph)

This repository is **not tied to any formal course or syllabus**.  
Its purpose is **conceptual understanding, experimentation, and demonstrations**.
